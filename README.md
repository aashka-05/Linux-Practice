# Linux Scripting .
Practicing Linux scripting as a part of my Devops learning journey.              

## To Setup - 
1. Clone the repository to your local device :
        ``` 
            git clone url
        ```
2. Navigate into the project folder :
        ``` 
            cd project_folder
        ```
2. Give execute permission to the script :
        ```
            chmod u+x filename.sh
        ```
3. RUN the script:
        ```
            bash filename.sh
        ```
