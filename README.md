#Practicing linux scripting.

##To setup:
1.Clone the repository to your local device : 
        ``` bash
            git clone url
        ```
2. Navigate into the project folder :
        ``` bash
            cd project_folder
        ```
2. Give execute permission to the script :
        ```
            chmod u+x filename.sh
        ```
3.run the script:
        ```
            bash filename.sh
        ```
